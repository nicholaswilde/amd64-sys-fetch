# Tech Stack: amd64-sys-fetch\n\n## Programming Language\n* **Assembly (x86_64 & aarch64):** Chosen for low-level control, high performance, and learning goals.\n* **Assembler:** [GNU Assembler](https://sourceware.org/binutils/docs/as/) (`as`), for its standard integration with the GNU toolchain.\n\n## Operating System & System Interface\n* **OS:** Linux (64-bit).\n* **Interface:** Direct Linux system calls (via `syscall` on x86_64 and `svc` on aarch64).\n* **Architecture:** 
    - **amd64:** Intel/AMD 64-bit instruction set (x64).
    - **arm64:** ARM 64-bit instruction set (AArch64).\n\n## Build & Tooling\n* **Linker:** GNU Linker (`ld`) for creating executable ELF64 binaries.\n* **Task Automation:** [Taskfile](https://taskfile.dev/) (Task) for managing build and execution workflows.\n* **Debugger:** `gdb` (GNU Debugger) for verifying register states and memory layout.
